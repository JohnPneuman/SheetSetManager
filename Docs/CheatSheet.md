# 🎹 Visual Studio Keyboard Cheatsheet

## 🔧 Algemene bewerkingen
| Actie                        | Sneltoets            | Opmerking                            |
|-----------------------------|----------------------|--------------------------------------|
| Commentaar toevoegen       | Ctrl + K, Ctrl + C   | Werkt op geselecteerde regels         |
| Commentaar verwijderen     | Ctrl + K, Ctrl + U   | Haalt // of <!-- --> weg              |
| Regel dupliceren           | Ctrl + D             | Dupliceert huidige regel              |
| Regel verplaatsen          | Alt + Up / Down      | Verplaatst regel omhoog/omlaag        |
| Regel verwijderen          | Ctrl + Shift + L     | Verwijdert hele regel                 |
| In-/uitzoomen              | Ctrl + Scrollwheel   | Zoomen in editor                      |

## 🔍 Navigatie
| Actie                        | Sneltoets            |
|-----------------------------|----------------------|
| Ga naar definitie          | F12                  |
| Peek definitie             | Alt + F12            |
| Ga terug                   | Ctrl + -             |
| Ga vooruit                 | Ctrl + Shift + -     |
| Ga naar symbool            | Ctrl + T             |
| Ga naar bestand            | Ctrl + ,             |

## ⚙️ Bouwen & starten
| Actie                        | Sneltoets            |
|-----------------------------|----------------------|
| Build project              | Ctrl + Shift + B     |
| Start met debuggen         | F5                   |
| Start zonder debuggen      | Ctrl + F5            |
| Stop debugging              | Shift + F5           |
| Herstart debugging          | Ctrl + Shift + F5    |

## 🐞 Debugging
| Actie                        | Sneltoets            |
|-----------------------------|----------------------|
| Zet breakpoint             | F9                   |
| Stap over (step over)      | F10                  |
| Stap in (step into)        | F11                  |
| Stap uit (step out)        | Shift + F11          |
| Ga naar volgende breakpoint| Ctrl + F9            |

## 🧹 Opruimen & herschikken
| Actie                        | Sneltoets            |
|-----------------------------|----------------------|
| Document formatteren       | Ctrl + K, Ctrl + D   |
| Namen wijzigen (refactor)  | Ctrl + R, Ctrl + R   |
| Snelle actie (lightbulb)    | Ctrl + .             |

---

> Tip: Deze sneltoetsen werken in Visual Studio 2019, 2022 en hoger. Je kunt ze aanpassen via **Tools → Options → Keyboard**.

Laat deze openstaan naast je project of pin 'm vast aan je dock voor snelle reference 🚀
