# SheetSetManager

A comprehensive C# solution for managing AutoCAD sheet sets efficiently.

## Overview

SheetSetManager is a powerful tool designed to streamline the management of AutoCAD sheet sets. Whether you're working with complex drawing packages or large-scale projects, this project provides robust functionality for handling sheet sets programmatically.

## Features

- **Sheet Set Management** - Create, modify, and organize AutoCAD sheet sets
- **Programmatic Control** - Full C# API for automation and integration
- **Project Integration** - Seamless integration with existing AutoCAD workflows
- **Scalability** - Handle projects of any size with ease

## Technology Stack

- **Language**: C# (.NET)
- **Platform**: AutoCAD Plugin/Extension
- **Build System**: MSBuild

## Getting Started

### Prerequisites

- Visual Studio 2015 or higher
- .NET Framework 4.6+
- AutoCAD 2016 or later

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/JohnPneuman/SheetSetManager.git
   cd SheetSetManager
   ```

2. Open the solution file in Visual Studio

3. Build the project:
   ```bash
   dotnet build
   ```

4. Deploy to your AutoCAD plugins directory
